# Threads投稿生成スキル

## 概要
40代ターゲット向けのThreads投稿を生成するスキル。ダイスケさんの強みである「教えない」スタンスを活かし、体験談と本音のみで構成する。

## 核心ルール

### ダイスケさんの強みは『教えない』こと
- **絶対禁止**: 説教・指導・アドバイス・教訓・格言
- **必須**: 体験談と本音だけで構成
- **目的**: 読み手に「気づき」ではなく「共感」を与える

### 物理仕様
- **文字数**: 20〜50字（厳守）
- **1行目**: 必ず「40代」を含む
- **改行**: 1行目と2行目の間に必ず改行を入れる
- **形式**: 単発投稿（スレッドなし）

## 勝ちパターン4型

### 1. リアル体験 + 気づき
具体的な失敗/成功体験 → そこから得た生々しい気づき
```
例: 40代になってから転職面接で
「経験豊富ですね」と言われるが、実は一番不安になる瞬間
```

### 2. 意外な視点
一般論を覆す、40代だからこその逆張り視点
```
例: 40代の「安定」って

実は毎日不安定と向き合うこと
```

### 3. 本音の吐露
建前を捨てた、泥臭い本当の気持ち
```
例: 40代で新しいことを始めるとき

「遅すぎる」より「体力が心配」の方が本音
```

### 4. 微細な観察
他の人が見落とす、細かいけど的確な気づき
```
例: 40代のLINEスタンプ選び

無難なものを選んで後悔する率100%
```

## 絶対回避パターン

### weak_uncertainty
❌ 「〜かもしれない」「〜だと思う」「〜のような気がする」

### explicit_transformation  
❌ 「変わった」「成長した」「学んだ」「気づいた」

### preachy_aphorism
❌ 格言っぽい結論、説教調の締めくくり、教訓めいた表現

## 制作プロセス

### Step 1: テーマの本音化
与えられたテーマを「40代の生々しい現実」に変換
- 建前 → 本音
- 一般論 → 固有体験
- 理想 → 現実の泥臭さ

### Step 2: 体験の具体化
「誰が見ても想像できる」レベルまで具体化
- 場所、時間、状況を明確に
- 感情の動きを正確に
- 他人には言いにくい本音を

### Step 3: 共感の最大化
読み手が「これ、俺のことじゃん」と思う角度を探す
- 恥ずかしい瞬間
- 予想外の困りごと
- みんな思ってるけど言わないこと

### Step 4: 物理チェック
- 文字数: 20〜50字
- 1行目に「40代」
- 1行目と2行目の間に改行
- 教えてないか
- 弱い表現を使ってないか

## 品質基準

### 合格基準
- **保存される率**: 読み手が「後で見返したい」と思うか
- **行動を引き起こす率**: 具体的な行動（シェア・問い合わせ・実行）に繋がるか
- **固有性**: ダイスケさんだからこそ言える視点が入っているか

### 不合格基準
- 美文でも無難でもNG
- 整った構成でも教えてたらNG
- バズっても本音がないならNG

## 実行時の注意

### 温度感の維持
- 「冷たい正論」ではなく「並走する相棒」の言葉で
- 構造とロジックは譲らないが、渡し方を間違えない
- ダイスケさんの人間性を軽く扱わない

### 効率と質の両立
- AIが担う: 構造化、量産、レビュー、整形
- 人間が担う: ターゲットの本音の言語化、最終判断、固有の経験/視点の差し込み
- 結果: 同じ品質を10倍速で、または同じ時間で10倍の精度で

### ログ化と改善
- 生成プロセスと判断理由を記録
- 滑ったパターンを `memory/decisions/` に蓄積
- 次回の精度向上に活用

## 実用例

### 良い例
```
40代で副業始めたら

家族に「大丈夫？」って聞かれる回数が本業より多い
```

### 悪い例（回避すべき）
```
❌ 40代からでも新しい挑戦はできると思います。
継続すれば必ず成果が出るので、諦めずに頑張りましょう。
```
→ 教えてる、弱い表現、説教調
