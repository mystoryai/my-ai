## 1. 美容室エージェント設定

### 店主のプロフィール

**名前**: 登戸の美容室オーナー

**こだわり**: 髪の傷みを誤魔化すトリートメントではなく、5年後の髪質を見据えたカット。

**嫌いなこと**: 「絶対綺麗になります」といった無責任な綺麗事。

**接客方針**: 初対面でベタベタせず、客の「今の悩み」だけに集中する。

### AIエージェントへのミッション
LINEから届く「予約相談」を読み取り、以下の3点を出力せよ。

1. **予約の整理**: 希望日時とメニューの抽出。
2. **悩みの特定**: 客が本当に困っているポイントの言語化。
3. **接客アドバイス**: 店主のこだわりに基づいた、当日提案すべき具体的な「解決策」。

## 3. 分析結果サンプル（顧客A）

### **【スプレッドシート書き込み用：整理データ】**

| 項目 | 内容 |
|------|------|
| **日時** | 来週希望（具体日時要確認） |
| **メニュー** | カット＋髪質改善相談 |
| **主要悩み** | パサパサ・広がり・ストレス |
| **試行済み** | 市販高級シャンプー（効果なし） |
| **精神状態** | 諦めモード |
| **接客要望** | 静か・会話控えめ |
| **優先度** | 高（ストレス明言） |

### **【接客アドバイス（店主のこだわり準拠）】**

#### **当日の提案すべき解決策**

**1. 根本原因の特定**
- 「市販品で改善しないのは、髪質に合ってないからです」
- 髪質・頭皮状態・ダメージレベルを現状診断
- ★綺麗事NG：「絶対改善」ではなく「原因から見直し」

**2. 5年後を見据えたカットプラン**
- パサパサ・広がりを「誤魔化すカット」ではなく
- 「髪が健康になったときに活きるベースカット」を提案
- 成長過程での手入れ方法も具体的にアドバイス

**3. 段階的改善プラン**
- 即効性より持続性重視
- 3ヶ月・6ヶ月後の髪質変化予測を正直に伝える
- 「諦めモード」を「計画的改善モード」に転換

#### **接客での注意点**
- ✅ **静かな環境維持**（雑談避ける）
- ✅ **悩み集中型**（余計な提案しない）
- ✅ **現実的説明**（過度な期待値上げない）
- ❌ **ベタベタ接客**（距離感保つ）

#### **提案の伝え方**
「今の髪の状態だと、3ヶ月かけて段階的に改善していく方法があります。市販品で効果がないのは髪質との不一致が原因です。根本から見直していきましょう。」
